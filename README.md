
# Awesome Books

> _Awesome books_ is a website that displays a list of books and allows users to add and remove books from the list.

## Built With

- HTML
- CSS
- JavaScript


## Getting Started

To get a local copy, run the fllowing command.


### Prerequisites

- node.js
- git

### Install

- `npm install `

### Usage
Run live server
